Coding Challenge

15 January 2021

Challenge:

Please create a function that takes in an array of numbers and returns a new array that only contains odd numbers.

Notes:

To see our odd array function working feel free to open up the console within your browser and call the "newArrOdd(arr)"" function with your own array as an argument. For example, to keep things simple try out: "newArrOdd([1, 2, 3, 4, 5, 6, 7, 8, 9])".
