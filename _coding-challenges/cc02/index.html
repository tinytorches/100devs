Coding Challenge

12 January 2021

Challenge:

Please create a function that takes in a number and a word. If the length of the word multiplied by the number passed into the function is greater than 100, alert "WINNER"!

Instructions:

  1. Enter "winnerWinnerChickenDinner(number, word)" in your browser console.
  2. Put in your own number for "number".
  3. Put in your own word for "word" wrapped in quotes.
  4. Press "Enter"
  5. Are you a winner?
